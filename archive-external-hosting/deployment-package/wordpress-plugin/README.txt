Embed your RareBeats music player/store on any WordPress page using shortcodes.

== Description ==

RareBeats Player allows you to easily embed your beat store and music player on any page of your WordPress site using simple shortcodes.

Features:
* Easy shortcode integration
* Filter beats by genre, mood, or BPM
* Fully responsive design
* WooCommerce integration
* Customizable height and width
* Multiple display styles

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/rarebeats-player`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings â†’ RareBeats Player to configure
4. Use shortcode [rarebeats_player] on any page

== Frequently Asked Questions ==

= How do I embed the player? =

Simply add [rarebeats_player] shortcode to any page, post, or widget.

= Can I filter by genre? =

Yes! Use [rarebeats_player genre="Trap"] to show only Trap beats.

= How do I change the height? =

Use [rarebeats_player height="1000px"] to set a custom height.

= Can I use it on multiple pages? =

Yes! You can use the shortcode on as many pages as you want.

== Shortcode Attributes ==

* height - Player height (default: 800px)
* width - Player width (default: 100%)
* genre - Filter by genre (Trap, Hip-Hop, R&B, etc.)
* mood - Filter by mood (Dark, Chill, Uplifting, etc.)
* style - Display style: full, compact, minimal
